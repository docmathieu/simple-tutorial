Create a new tutorial

  1. Root folder

    2. To create an new tutorial, go to simple-tutorial/data folder. Copy and paste "simple-tutorial" folder.
    rename the new folder to "new-tuto".

    Inside the root folder:
    en contains images and pages for the English language
    fr contains images and pages for the French language
    setting.ini contains tutorial settings, for example the default language
    style.css overrides the global style

    Note that the 'en' and 'fr' folders might as well be called 'serialkiller', in this case you will see 'serialkiller' in the list of languages.

  2. Default language

    3. Setting the default language.
    At the same time we will remove facebook comments, we will see these confurations later.
    Modify setting.ini like this: 

    defaultLanguage=en
facebookComments=false
facebookID=

  3. Tutorial title

    4. The title is based on the language, so we have a setting.ini file in each language folder.
    Open this file simple-tutorial/data/new-tuto/en/setting.ini and change the title. 

    title=New tutorial

  4. Test the website

    5. In a new tab open http://tuto.mywebsite.com
    You can see a new link to your tutorial "New tutorial".
    Click on this link, the current data are those of the tutorial "Simple Tutorial", so we will delete them.
    First delete fr folder: simple-tutorial/data/new-tuto/fr

    Delete all files in the following folders:

    • simple-tutorial/data/new-tuto/en/images
    • simple-tutorial/data/new-tuto/en/pages

    There is no data in your tutorial so you will get an error if you reload the page.

  5. Recurrent content

    6. the container.html file is used to add recurrent HTML, which will be visible on all pages of the tutorial.
    This can be useful to display the name of the author, for example.

    Open this file simple-tutorial/data/new-tuto/fr/container.html and edit the content by: 

    [DATA]

    
    
    Author: Full name
    
    
    Date: xx-xx-xxxx

    In the next chapter we will add a new page.
