Add Comments

Simple Tutorial use Facebook Comments Plugin, so first we have to get our Facebook App ID.
To do this:

  1. click on the following url: https://developers.facebook.com/apps/
  2. click on the Add a New App button.
  3. Enter a name for your application eg "my tutorials" and click Create New Facebook App ID button.
  4. Select a category and click Create New Facebook App ID button.
  5. In "Setup the Facebook SDK for JavaScript", locate the appId and copy it.
  6. In "Tell us about your website", enter the url of your website for example http://tuto.mywebsite.com and click the next button.


You can now edit the file setting.ini 

defaultLanguage=en
facebookComments=true
facebookID=[paste your appId here]

WARNING !

In the chapter "Create a new tutorial" we said "Note that the 'en' and 'fr' folders might as well be called 'serialkiller'".
This statement is true but the language 'serialkiller' will not be recognized by Facebook comments.

simple-tutorial/data/locales.json file used to set the language codes.
If you really want a folder named 'serialkiller' with the English language, you can add this to locales.json: {"folder":"serialkiller", "code":"en_US"}
Here is the list of language codes available: https://www.facebook.com/translations/FacebookLocales.xml.
The code that interests us is in the representation tag.


Reload your page, you should see a comment block at the bottom of the page.
