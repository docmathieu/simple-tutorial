Simple Tutorial utilise Facebook Comments Plugin, il nous faut donc dans un premier temps récupérer un App ID Facebook.
Pour ce faire:

  1. Cliquez sur l'url suivante: https://developers.facebook.com/apps/
  2. Cliquez sur le bouton Add a New App.
  3. Donnez un nom à votre application, par exemple "mes tutoriels", et cliquez sur Create New Facebook App ID.
  4. Sélectionnez une catégorie et cliquez sur Create New Facebook App ID.
  5. Dans la partie "Setup the Facebook SDK for JavaScript", repérez l'appId et copier le.
  6. Dans la partie "Tell us about your website", renseignez l'url de votre site par exemple http://tuto.monsite.com et cliquez sur le bouton next.


Vous pouvez maintenant modifier le fichier setting.ini 

defaultLanguage=fr
facebookComments=true
facebookID=[collez votre appId ici]

ATTENTION !

Dans le chapitre "Créer un nouveau tutoriel" nous disions: "Notez que les dossiers 'en' et 'fr' pourraient tout aussi bien s'appeler 'unquoi'".
Cette affirmation reste vrai mais la langue 'unquoi' ne sera pas reconnue par facebook comments.

Le fichier simple-tutorial/data/locales.json sert à configurer les codes de langues.
Si vous souhaitez vraiment un dossier 'unquoi' avec la langue française, vous pouvez ajouter ceci au fichier locales.json:
{"folder":"unquoi", "code":"fr_FR"}
Voici la liste des codes langue disponibles: https://www.facebook.com/translations/FacebookLocales.xml.
Le code qui nous interresse se trouve dans la balise representation.
