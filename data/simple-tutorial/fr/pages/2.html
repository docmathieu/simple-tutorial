Créer un nouveau tutoriel

  1. Arborescence

    2. Pour créer votre tutoriel rendez-vous dans le dossier simple-tutorial/data.
    Copiez collez le dossier "simple-tutorial".
    Renommez le nouveau dossier en "new-tuto".

    Arborescence de notre dossier:
    en contient les images et les pages pour la langue Anglaise
    fr contient les images et les pages pour la langue Française
    setting.ini contient les configurations du tutoriel, par exemple la langue par défaut
    style.css permet de surcharger le style global

    Notez que les dossiers 'en' et 'fr' pourraient tout aussi bien s'appeler 'unquoi', dans ce cas vous verriez 'unquoi' dans la liste des langues.

  2. Langue par défaut

    3. Configurons la langue par défaut.
    Dans le même temps nous retirons les commentaires facebook, nous verrons ces confurations plus tard.
    Ouvrez le fichier setting.ini et modifiez le pour obtenir ceci: 

    defaultLanguage=fr
facebookComments=false
facebookID=

  3. Titre du tutoriel

    4. Le titre est fonction de la langue, nous avons donc un fichier setting.ini dans chaque dossier de langue.
    Ouvrez le fichier simple-tutorial/data/new-tuto/fr/setting.ini et modifiez le title. 

    title=Nouveau tutoriel

  4. Tester le site

    5. Dans un nouvel onglet affichez votre page http://tuto.monsite.com
    Vous pouvez constater un nouveau lien vers votre tutoriel "Nouveau tutoriel".
    Cliquez sur ce lien, les données actuelles sont celles du tutoriel "Simple Tutoriel", nous allons donc les supprimer.
    Commencez par supprimer le dossier concernant l'Anglais: simple-tutorial/data/new-tuto/en

    Supprimez tous les fichiers des dossiers suivants:

    • simple-tutorial/data/new-tuto/fr/images
    • simple-tutorial/data/new-tuto/fr/pages

    Il n'y a plus de données dans votre tutoriel vous aurez donc une erreur si vous rechargez la page.

  5. Contenu récurrent

    6. En naviguant dans les dossiers vous aurez peut-être remarqué des fichiers nommés container.html
    Ces fichiers permettent d'ajouter du HTML récurrent, qui sera visible sur toutes les pages du tutoriel.
    Cela peut-être utile pour afficher le nom de l'auteur par exemple.

    Ouvrez le fichier simple-tutorial/data/new-tuto/fr/container.html et modifiez en le contenu par: 

    [DATA]

    
    
    Auteur: Prenom Nom
    
    
    Date: xx/xx/xxxx

    Dans le chapitre suivant nous ajouterons une nouvelle page.
